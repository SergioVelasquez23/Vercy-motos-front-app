# Script para generar el nuevo logo de Sopa y Carbón

## Necesitamos reemplazar assets/images/logo.png con un nuevo diseño

### Características del nuevo logo:

- Tamaño: 512x512px (alta resolución)
- Fondo: Naranja corporativo (#FF6B00)
- Diseño: Logo del restaurante "Sopa y Carbón"
- Formato: PNG con transparencia

### Pasos para solucionarlo:

1. Reemplazar assets/images/logo.png con el nuevo logo
2. Ejecutar flutter packages pub run flutter_launcher_icons
3. Rebuild y redeploy

### Alternativa rápida:

Como no tengo acceso directo para crear/editar imágenes PNG,
voy a generar los iconos directamente usando código.
